Maps

Linear Admin uses JSvectormap to meet all of its mapping requirements. Jsvectormap is a lightweight JavaScript library for creating interactive maps and pretty data visualization, supports all modern brownsers including IE9+.

Plugin documentation: https://github.com/themustafaomar/jsvectormap

Basic usage

Simply include the following HTML and JS code along with the map file to render the map in you web page.

<div id="map-world"></div>

import jsVectorMap from "jsvectormap";
import "jsvectormap/dist/maps/world.js";

new jsVectorMap({
    selector: "#map-world",
    map: "world",
    draggable: false,
    zoomButtons: false,
    zoomOnScroll: false,
    regionStyle: {
        initial: {
            fill: "#cccccc",
            "fill-opacity": 1,
            stroke: "none",
        },
    }
});
