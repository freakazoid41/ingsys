Horizontal

The collapse plugin supports horizontal collapsing. Add the .collapse-horizontal modifier class to transition the width instead of height and set a width on the immediate child element.

Multiple toggles and targets

A <button> or <a> element can show and hide multiple elements by referencing them with a selector in its data-bs-target or href attribute.
